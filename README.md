# Калькулятор цен артефактов и частей мутантов

Этот сайт помогает быстро считать стоимость игровых предметов. Он нужен для удобного подсчёта цены артефактов, частей мутантов и сигарет.

## Что есть на сайте

- Калькулятор артефактов.
- Калькулятор частей мутантов.
- Калькулятор сигар.
- Поиск по предметам.
- Переключение светлой и тёмной темы.
- Редактирование цен.
- Экспорт текстового отчёта.

## Как пользоваться

1. Открой нужный раздел.
2. Найди предмет через поиск или вручную.
3. Нажимай на карточку предмета или кнопку `+`, чтобы добавить количество.
4. Используй кнопку `-`, чтобы уменьшить количество.
5. Выбери процент наценки или скидки, если он нужен.
6. Смотри базовую сумму и итоговый результат сверху.

## Быстрое управление

- Клик по иконке предмета: добавить `1` штуку.
- `Shift + клик` или `ПКМ`: убрать `1` штуку.
- Средняя кнопка мыши по карточке: сбросить количество у конкретного предмета.
- Двойной клик по цене: изменить цену прямо на карточке.
- Клик по сумме: скопировать число.

## Настройка цен

В разделе артефактов есть кнопка настроек, через которую можно изменить цены. Для некоторых позиций цену также можно менять двойным кликом по самой цене.

Изменённые значения сохраняются в браузере, поэтому после перезагрузки страницы они не пропадают.
